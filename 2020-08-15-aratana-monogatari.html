我想這大概已經…做好了吧！
建立一個網站可真的不容易…雖然以前用WordPress建過，但是那是給小孩子玩的東西，完全傻瓜操作。有後端和add-on，超級方便。
這次確實是想嘗試一點新的東西，所以大部分工作都在CLI下進行，對我這種擊敗人非常不友好。畢竟已經被圖形介面慣壞了。

那麼……稍微總結一下？

一開始我是在heroku apps上嘗試部署的，可是…到最後push不上去…現在我大概知道了原因。
在執行這條指令 $ git push heroku master時，沒有配置遠端實際在使用指令heroku create時就已經配置好了遠端，代替了git remote add的操作，於是出現錯誤
error: src refspec master does not match any error: failed to push some refs to '（我的heroku倉庫？好我也不知道怎麼叫）
實際上是我忘記在git clone node-js-getting-started之後cd到那個位置。

然而在除錯之前我就放棄了heroku轉用GitHub Pages。也出現很多問題，比如SSH金鑰、檔案位置問題、_config.yml的設定問題。
幸運的是這些問題都可以查閱官方檔案解決，很方便方便到我連續做了十幾個小時網站才上線，這裡不再贅述了。

終於，在今天午前1時左右，上線了！
一次新的嘗試，然後這裡便要開始新的故事。

我計畫就在這裡寫一些日常，緩慢更新，技術類文章估計很少。

好的，到此為止。
